# Роутинг и страницы (views)

#code #router #views

**Файл маршрутов:** `mykiddy/App.tsx`.

---

## Обёртки

| Компонент | Условие |
|-----------|---------|
| `ProtectedRoute` | Не гость; иначе `<Navigate to="/" />` |
| `StaffRoute` | Не гость **и** `user.role === Admin`; иначе `/` |

Пока `isLoading` — полноэкранный лоадер (бренд «Дети В ТОПЕ»).

---

## Таблица роутов

| Path | Компонент | Защита | Назначение |
|------|-----------|--------|------------|
| `/` | `Dashboard` | нет (гость видит урезанно) | Главная, курсы, стрик, ссылка на витрину |
| `/courses` | `CourseDetail` | Protected | Уроки, видео, ДЗ, AI-проверка, театр видео |
| `/profile` | `Profile` | Protected | Профиль, витрина-модалка, настройки профиля |
| `/settings` | `Settings` | Protected | Настройки аккаунта |
| `/notifications` | `Notifications` | Protected | Лента `activity_notifications` |
| `/schedule` | `Schedule` | Protected | Расписание из `schedule_events` |
| `/community` | `Community` | Protected | Люди + вкладка витрины `?v=showcase` |
| `/users/:userId` | `UserPublicProfile` | Protected | Публичный профиль, витрина пользователя |
| `/admin` | `AdminPanel` | StaffRoute | Курсы, юзеры, расписание, модерация витрины |
| `*` | редирект на `/` | | |

**AI-тьютор и проверка ДЗ** встроены в **`CourseDetail`** (не отдельный URL).

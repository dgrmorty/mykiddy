# Сервисы и интеграции

#code #services

**Папка:** `mykiddy/services/`.

---

## supabase.ts

- Единственный клиент `createClient(VITE_SUPABASE_URL, VITE_SUPABASE_ANON_KEY)`.
- Сессия и `onAuthStateChange` для синхронизации с `AuthContext`.

---

## contentService.ts

- **`getCourses(userId?)`**: выборка `courses` + вложенные `modules` + `lessons`, порядок по `created_at`.
- Мерж с **`user_progress`** для отметок завершения.
- Кэш в памяти ~90 с; **`invalidateCoursesCache`** после админских правок контента.
- Таймаут запроса ~8 с → **`CoursesLoadError`** для UI «Повторить».

---

## geminiService.ts

- **`getApiUrl`** из `config.ts` для всех вызовов.
- **`askAiTutor`**, **`checkHomework`**, вложения **`HomeworkAttachment`**, **`getAiUsageQuota`**, **`checkSystemHealth`**.
- Обработка 429/401 и сетевых ошибок.

---

## projectShowcaseService.ts

- CRUD витрины: **`project_posts`**, лайки, загрузка в Storage bucket **`project_showcase`**.
- **`fetchApprovedShowcasePosts`**, fallback RPC **`list_approved_showcase_posts`**.
- **`fetchShowcaseAuthorsForFeed`** → RPC `list_showcase_authors` + fallback `profiles`.

# React-контексты

#code #react #context

**Расположение:** `mykiddy/contexts/`.

---

## AuthContext

- **Файл:** `AuthContext.tsx`
- **Состояние:** текущий `User`, `isGuest`, `isLoading`, модалка входа.
- **Профиль:** загрузка из `profiles` (включая `streak_*`, `avatar_accessory`), merge аватара с bundled AI-аватарами (`data/defaultAvatars.ts`).
- **Стрик:** RPC `record_daily_streak` после успешного чтения профиля (с проверкой сессии).
- **Методы:** `requireAuth`, `refreshUser`, `openAuthModal`, `signOut`.

---

## ContentContext

- **Файл:** `ContentContext.tsx`
- **Состояние:** `courses`, `activeCourse`, `activeLesson`, загрузка/ошибки.
- **Источник данных:** `contentService.getCourses`, инвалидация кэша при необходимости.

---

## BrandingContext

- **Файл:** `BrandingContext.tsx`
- Школа: `settings` в Supabase (`logo_url`, `school_name`).

---

## ToastContext

- Глобальные уведомления (успех/ошибка/info).

---

## NotificationContext

- **Файл:** `contexts/NotificationContext.tsx`
- Уведомления из **`activity_notifications`**, счётчик непрочитанных для Layout/Sidebar.

---

## GuidedTour / onboarding

- `GuidedTourContext`, компонент **`components/onboarding/OnboardingTour`** — привязка к `data/onboardingTour.ts`.

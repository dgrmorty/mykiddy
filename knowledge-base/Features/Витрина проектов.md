# Витрина проектов
#feature #community

## Что это
Раздел сообщества где ученики публикуют свои проекты — то что они сделали на курсах. Мотивирует через социальное признание.

## Функционал
- Публикация проекта: название, описание, ссылка на проект
- Лента постов других учеников
- Доступно только авторизованным (`ProtectedRoute`)

## БД
Таблица `community_posts`: user_id, title, content, project_url, created_at

## Компонент
`views/Community.tsx`

# Расписание
#feature #schedule

## Что это
Еженедельное расписание занятий школы. Ученики видят когда и где их уроки.

## Структура данных
```typescript
ScheduleEvent {
  day_of_week: number  // 1=Пн, 2=Вт, ... 7=Вс
  time_start: string   // "10:00"
  time_end: string     // "11:30"
  title: string        // "Python Мастер — группа A"
  location: string     // "Zoom / Аудитория 3"
}
```

## Управление
Создаётся и редактируется в `AdminPanel` → только ADMIN.

## Компонент
`views/Schedule.tsx`

# Профиль и геймификация
#feature #gamification #profile

## Профиль ученика
- Аватар, имя, роль
- Уровень + XP прогресс-бар
- Стрик (текущий и лучший)
- Радар-график навыков

## Радар навыков (Recharts)
6 осей: Логика, Синтаксис, Архитектура, Отладка, Креатив, Скорость
Данные берутся из `SKILL_DATA` (constants.ts) или из Supabase

## Публичный профиль
Роут `/users/:userId` → `views/UserPublicProfile.tsx`
Виден другим ученикам — социальный элемент

# AdminPanel
#feature #admin

## Доступ
Только `Role.ADMIN`. Защищён `StaffRoute` в `App.tsx`.
Список admin email-ов → `VITE_ADMIN_EMAILS` в env.

## Возможности
- **Курсы**: создание, редактирование, удаление course/module/lesson
- **Пользователи**: список всех (`get_all_users` RPC), удаление (`delete_user_by_admin` RPC)
- **Расписание**: CRUD событий `schedule_events`
- **Настройки**: брендинг школы (название, цвета, логотип) через таблицу `settings`

## ⚠️ Безопасность
Запросы идут с клиента с JWT → Supabase RLS проверяет роль.
Без правильных RLS-политик любой авторизованный юзер может вызвать те же методы из консоли!

## Компонент
`views/AdminPanel.tsx`

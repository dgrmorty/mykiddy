# Видеоуроки
#feature #core

## Что это
Основная учебная единица платформы. Каждый урок содержит видео (YouTube), опциональный блок кода и задание для домашней работы.

## Иерархия
```
Course → Module[] → Lesson[]
```

## Поля урока
- `videoUrl` — ссылка на YouTube
- `codeSnippet` — блок кода для практики (язык по типу курса)
- `homeworkTask` — текст задания ДЗ
- `isCompleted` — синхронизируется с `user_progress` в Supabase
- `locked` — урок недоступен пока не пройден предыдущий
- `upcomingDate` — дата когда откроется будущий урок

## Прогресс
- Отмечается в `user_progress` (user_id + lesson_id + completed_at)
- За завершение урока начисляется XP → [[Система прогрессии]]
- Прогресс модуля = completed / totalLessons * 100

## Компоненты
- `views/CourseDetail.tsx` — страница курса
- `contexts/ContentContext` — загрузка и кэш курсов

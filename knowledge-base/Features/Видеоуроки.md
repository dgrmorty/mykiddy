# Видеоуроки и курс

#feature #core #courses

---

## Иерархия данных

```
Course (courses)
  └── Module (modules) — sort_order
        └── Lesson (lessons) — sort_order
```

Типы курсов: см. **`CourseType`** в `types.ts` (Python Мастер, Web Архитектор, …).  
Год обучения: **`year_tier`** — `year_1` / `year_2_plus` (`normalizeCourseYearTier`).

---

## Поля урока (суть)

| Поле | Назначение |
|------|------------|
| `video_url` | Ссылка (часто YouTube) |
| `code_snippet` | Практика в редакторе |
| `homework_task` | Текст ДЗ |
| `locked` | Блокировка цепочкой |
| `upcoming_date` | Текст даты для «скоро» |

На клиенте в типе **`Lesson`** поля могут маппиться в camelCase при сборке в `contentService`.

---

## Прогресс

- Таблица **`user_progress`**: `user_id`, `lesson_id`, `completed_at`.
- **`contentService`** мержит completed lesson ids в объект курсов для UI.
- Завершение урока — запись в `user_progress` + начисление XP (логика в `CourseDetail` / сервисах).

---

## UI

- **`views/CourseDetail.tsx`**: выбор урока, видео (в т.ч. theater mode), ДЗ, AI-тьютор, проверка ДЗ.
- **`contexts/ContentContext`**: активный курс/урок для навигации.

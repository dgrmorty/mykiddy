# Проверка домашних заданий (AI)

#feature #homework #gemini

---

## Где в продукте

- Страница **`CourseDetail`** (`views/CourseDetail.tsx`): модалка сдачи ДЗ, текст + вложения (изображения/видео с лимитами размера).
- Вызов **`checkHomework`** из **`services/geminiService.ts`** → **`POST /api/check-homework`** на Express.
- Ответ наставника кешируется в **`localStorage`** по ключу урока/пользователя (см. код) для повторного показа.
- Логика «засчитано/на доработку»:** `utils/homeworkAcceptance.ts`**, форматирование текста:** `utils/formatAiFeedback.ts`**.

---

## Сервер

- Лимит запросов по дню: **`AI_DAILY_HOMEWORK_LIMIT`**, учёт в **`ai_usage`**.
- JWT обязателен для нормальной квоты (см. `server.js`).

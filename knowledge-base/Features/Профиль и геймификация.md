# Профиль и геймификация

#feature #profile #xp #badges

---

## Профиль

- **`views/Profile.tsx`**: XP, уровень, аватар (в т.ч. bundled из `data/defaultAvatars.ts`), аксессуар **`avatar_accessory`**, модалка подачи проекта на витрину.
- Обновление профиля через RPC **`update_own_profile_patch`** и/или прямой patch (см. актуальный `AuthContext` / Profile).

---

## Прогрессия

- **`progression.ts`**: `XP_PER_LEVEL = 500`, `levelFromXp`, `xpLevelProgressPercent`.
- Стрики: поля **`streak_current`**, **`streak_longest`**, **`streak_last_activity_date`**, RPC **`record_daily_streak`** (UTC).

---

## Бейджи

- **`hooks/useBadgeProgress.ts`**, каталог **`data/badgeCatalog.ts`**, UI **`components/BadgeOrb`**, модалка выбора аватара.

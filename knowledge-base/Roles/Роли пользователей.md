# Роли пользователей

#roles #auth

| Роль | Enum | Доступ (обзор) |
|------|------|----------------|
| Student | `Role.STUDENT` | Защищённые роуты, курсы, профиль, сообщество, витрина |
| Parent | `Role.PARENT` | Сценарии просмотра прогресса (как задумано в продукте) |
| Teacher | `Role.TEACHER` | Расширенный доступ к контенту/данным по политикам БД |
| Admin | `Role.ADMIN` | Всё + **AdminPanel** (`/admin`) |
| Guest | `Role.GUEST` | Только главная без входа в закрытые разделы |

---

## Admin

- Список email: **`VITE_ADMIN_EMAILS`** (через запятую).
- Дополнительно нормализация в **`AuthContext`**; не полагаться только на фронт — **RLS** в Supabase.

---

## Guards (`App.tsx`)

| Обёртка | Условие доступа |
|---------|-----------------|
| `ProtectedRoute` | Не гость → иначе редирект на `/` |
| `StaffRoute` | `user.role === Admin` → иначе `/` |

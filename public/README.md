# Папка `public`

Сюда кладите **статические файлы**: они копируются при сборке и доступны по корню сайта.

**Примеры:**
- `public/favicon.ico` → на сайте как `/favicon.ico`
- `public/laptop.glb` → на сайте как `/laptop.glb`

Сейчас в проекте загрузка GLB из этой папки отключена (используется процедурная 3D-модель ноутбука). Папка может быть пустой — приложение от этого не ломается.
